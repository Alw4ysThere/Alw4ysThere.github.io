杨德昌留给世界的财富——《一一》

 首先感谢B站up主陈小姐的十五楼，从她的作品中我收获了很多很多，这篇博客可以说绝大部分内容来自与她对于《一一》的解读。

什么是“一一”
#

 在看电影的时候，我认为“一一”就是一天一天的生活，一代又一代的传承，整体上电影记录了NJ一家人的生活，妻子说一天天的过着相同的日子，已经走向了崩溃，父/女/子一代人一代人之间也处于一个循环状态，经历着类似的情感生活。
 我其实非常能够共情这位母亲，我自从每天写日记开始，就发觉大部分时间我都是过着一成不变的，类似的日子，遇到有一天可以写不一样的事情，我就会很开心。我有时候在重复的日子中，我也会拷问自己，生命的意义在哪里，我学了这么多东西，我到底相比别人，独特在何处？尽管我读了很多加缪，但是即使读了再多，我还是觉得存在主义的荒诞与虚无感受，永远是现代社会的人无法跨越的槛。


 陈小姐则告诉我“一一”是个体的符号，影片中两个一平行放置，意味着现代社会独立不重合的个体相遇，一生二，二生三，三生万物，一个个个体形成了我们的社会与文明。
 进入现代社会，摆在我们面前的一个重大问题，就是如何调和人与人，主体与主体之间的相互关系。
 在初中的时候，我谈过一个女朋友。最终我们不欢而散，老死不相往来，我觉得在这段感情之中，我把他当作了附属品。其实直到今天，也是如此，谈恋爱就意味付出，意味无处不在的商讨与妥协。我不喜欢。我更喜欢随心所欲的生活，有时候甚至畏惧恋爱。我心目中的恋爱，只是希望有人听我倾诉发表想法，陪我经历各种事情，去往各种地方，但是我从来没有想过，我要去参与她的生活。《春物》说得好，两个人人生的交汇，其实是互相扭曲，互相干涉彼此人生的过程，我害怕这样的过程，始终踏不出这一步。或许我还没有为恋爱做好准备。
 有的时候觉得，自己是如此矛盾，我渴望多彩的，不一样的人生，却在舒适区里面犹犹豫豫，就这样过一辈子。

如何解决主体之间的矛盾？
#

 我是我自己，我是父母的孩子，未来我可能会成为谁谁谁的男友，丈夫，父亲，如何处理自己和他们的关系？
 一言以蔽之，看清楚自己所处的社会位置，用生命情感安放自己同他人的之间的关系。
 NJ对他的初恋说，他只爱过她一个人，对妻子孩子，他用肩膀扛起这个家的责任，说到底，人在社会关系中保持绝对的自主还是不存在的，重要的还是真诚的对待他人，同时培养艺术人格，像NJ一样，可以是在音乐中，可以是在文学中，在一个自己喜欢的地方，温养自己的内心。
 我们现在与他人交往的行为似乎成为了一种策略行为，而不是主题情感行为，我们以自我为中心，为了利益与他人去交往，现代社会将此视为一种能力，一种“本事”，我觉得还是有些可悲。
 NJ这样的是好人啊，可是好人是会被现代都市淘汰的。无论是婆婆，还是发觉生命缺乏意义的母亲，服务前女友吸煤气的阿弟，最后到病倒的NJ，无论是傻傻的蠢人，还是大大的好人，想要处理好人与人直接的关系是很累的，是耗费精力的，我们难以抵挡这样的攻击，最后是坏人留在了舞台之上。
 人是一切社会关系的总和，这句话越看越对，越经历越发觉它是真理。

两段BE的情感
#

 影片讲述了三段情感。一段是小男孩初生的情愫，他为了“小老婆”练习憋气，学会了游泳，看到他的表情，真是发自内心的快乐。
 剩下两段则要失意一些。NJ遇到了他的初恋女友，两人共赴日本，重温过往约会的日子。当初是男主不辞而别，甩了女方，这次他说了这么一段话：
 人是不可能让另外一个人去教他怎么活下去，怎么过日子，那是很悲哀的，你知道吗？
 当时他的父母，女友，都希望他去读电机系的博士，觉得这样才能够赚钱，有出路，但是NJ却不喜欢。这就是当代人的自我主体意识，无论是多么亲密的关系，我们都不喜欢别人干涉自己的人生，想要自己的路自己走。过往的日子哪有这些事情，古代的时候你要和谁结婚都安排的明明白白的，少数反抗的人最后哪个有好下场的。
 NJ的女儿婷婷在闺蜜和他的男友“胖子”闹掰了之后，成为两人之间交流的桥梁，不过这时候胖子对他出手了，这对于情感晚熟的婷婷来说完全是一记霹雳。这里是全片的一个名场面，父亲，女儿，东京，台北，各自进行着各自的约会，经历着各自的情感，走过路口，牵起汗津津的，对方的手，交叉剪辑，似乎下一秒就是两代人的遇见。


 很可惜，在过去，女孩带着NJ去开房，NJ逃跑了，这次婷婷鼓起勇气，带着胖子去开房，胖子心里念着的还是前女友，他也跑路了，有点好笑。爱情的问题，往往最终的解答就是金庸在《白马啸西风》中所说的，这些那些都很好，可是就是自己不够爱。
 说了这么多，我在想，如果回到过去，我和她的结局会有什么改变。我想说，恐怕真的会不一样,我这样期望着，期望着。

人永远看不见自己的后脑勺
#

 人看不见自己的后脑勺，人对于自己永远有一半是未知的，必须要通过他人来照见。就像NJ遇见大田，与他人的关系帮助我们认清一个完整的自己。

电影的绝美结局
#

 电影中的小男孩简洋洋最后说了这么一段台词，代表了导演杨德昌对于人类未来的美好祝愿，我想也是他作为一位电影人的初心使命,送给读到这篇博客的各位，如果有人在读的话：
 婆婆，对不起，不是我不喜欢跟你讲话，只是我觉得我能跟你讲的，你一定老早就知道了，不然，你就不会每次都叫我“听话”。就像他们都说你走了，你也没有告诉我，你去了哪里，所以，我觉得那一定是我们都知道的地方。婆婆，我不知道的事情太多了，所以，你知道我以后想做什么吗？我要去告诉别人他们不知道的事，给别人看他们看不到的东西，我想，这样一定天天都很好玩。说不定，有一天，我会发现你到底去了哪里，到时候哦，我可不可以跟大家讲，找大家一起过来看你呢？婆婆，我好想你，尤其是我看到那个还没有名字的小表弟，就会想起你常跟我说：你老了。我很想跟他说：我觉得……我也老了。


 如此这般，深刻的电影，优美流畅而又有一点残酷的故事，感谢您的宝贵时间。
